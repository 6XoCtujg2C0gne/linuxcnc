= simulate_probe(1)

== NAME

simulate_probe - simulate a probe input

== SYNOPSIS

*simulate_probe*

== DESCRIPTION

*simulate_probe* Creates an on-screen GUI button to simulate touch probe
input. Typically used in sim configs or debugging.
